# Fan-driver-with-thermistor
The circuit operates as a thermostat, turning the fan on and off to cool electronic components, thus ensuring stable operating conditions for the monitored device. The threshold for activating the system is set by the user - the voltage obtained from the divider with thermistor RT1 is compared with the reference voltage regulated by potentiometer PR1. When the resistance of the RT1 thermistor decreases below the value causing a voltage at the amplifier input lower than the value set at the input, the amplifier output will switch on the fan via the T1 transistor. When the temperature decreases, the forced cooling will switch off. Project designed and created during electronics lessons at university.
